# 🧠 Neurodiv Study Hub

Application d’aide au devoir pour étudiants neurodivergents (TDAH, dyslexie, TSA).  
Ce projet vise à offrir un espace numérique inclusif pour faciliter la concentration, l’organisation et la réussite scolaire.

---

## 🚀 Fonctionnalités principales

- 📚 **Gestion des tâches** : création, suivi et priorisation des devoirs.
- 🧩 **Mode focus** : interface simplifiée pour réduire les distractions.
- 🔔 **Rappels intelligents** : notifications adaptées au rythme de chaque utilisateur.
- 💬 **Espace collaboratif** : partage de conseils et d’astuces entre étudiants.
- 🌈 **Personnalisation** : thèmes visuels et outils adaptés aux besoins spécifiques.

---

| Composant | Description |

| **HTML / CSS / JS** | Interface utilisateur |
